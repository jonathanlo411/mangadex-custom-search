Project Details

This is a small webapp used to search MangaDex. The goal of this project is to provide an easy way to discover new Manga/Manhwa. As such there are bonus filtering options like including a minimum MyAnimeList score. This project utilizes the MangaDex API v3 and MyAnimeList API v2 Beta. I do not take any credit for the content or functionality from these two.
